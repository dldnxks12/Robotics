### Gym Environment Setting 

`Anaconda Package Finder` : https://anaconda.org/

- Install gym 

```commandline

conda create -n py3.6 python=3.6

conda activate py3.6

conda install -c conda-forge gym

conda install -c conda-forge gym-box2d

```

- Virtual Env Path

```commandline
conda env list
```
